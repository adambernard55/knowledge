# Checklist: Semantic Depth Content Audit

## Instructions

Use this checklist to audit existing content on a recurring basis (e.g., quarterly). The goal is to score each piece of content against the [[kb/SEO/8_frameworks-and-sops/01_semantic-depth-standard|Semantic Depth Standard]] and generate an actionable plan for improvement. This process is critical for building core update resilience.
